# ARRO

Price intelligence tools for New Zealand. We aggregate retail prices across the country so Kiwis can find the best deal without the legwork.

## Products

- **[Liquorfy](https://liquorfy.co.nz)** — Compare liquor prices across NZ retailers
- **Trolle** (coming soon) — Cross-supermarket grocery price tracking
- **Talle** (coming soon) — Multi-vertical price intelligence covering tech, home & appliances, pharmaceuticals, supplements, beauty, and pet goods

## Web

The marketing site lives in `web/` and is built with:

- React 19
- Tailwind CSS 4
- Vite
- Deployed on Vercel

### Development

```bash
cd web
npm install
npm run dev
```

### Build

```bash
cd web
npm run build
```
